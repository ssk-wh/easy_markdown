# Linux 深色主题自动检测

## 背景
Windows 已能跟随系统深色/浅色模式切换主题，Linux 上目前仅靠手动设置。需要根据桌面环境（GNOME / KDE）的主题信号自动检测。

## 动作
- [ ] GNOME：通过 `gsettings get org.gnome.desktop.interface color-scheme` 检测
- [ ] KDE：读取 `~/.config/kdeglobals` 的 `ColorScheme`
- [ ] 其他 DE：回退到 Qt 的 `QPalette` 启发式判断
- [ ] 监听变化（DBus 或文件 watch）实现实时切换
- [ ] 提供菜单项允许手动覆盖

## 验收
- GNOME 切换 color-scheme 后应用实时跟随
- KDE 切换 ColorScheme 后应用实时跟随
- 其他 DE 至少能在启动时正确判断深/浅
- 手动覆盖优先级高于自动检测
