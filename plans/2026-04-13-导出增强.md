# 导出增强：Word/EPUB/LaTeX 与自定义 CSS

## 背景
当前仅支持 PDF / HTML 导出。用户期望导出为 Word（docx）、EPUB、LaTeX 以满足学术、出版、办公场景，并希望能用自定义 CSS 控制 HTML/PDF 样式。

## 动作
- [ ] 评估候选库：`pandoc`（外部进程） vs 自实现转换
- [ ] 文件 → 导出 子菜单增加 Word / EPUB / LaTeX 三项
- [ ] 实现导出管道：Markdown → AST → 各目标格式
- [ ] 自定义 CSS：导出对话框中可选择本地 .css 文件，写入 HTML/PDF 渲染时套用
- [ ] 导出失败时弹出明确错误信息

## 验收
- 三种新格式均能导出包含标题/列表/代码块/表格/图片的样例文档
- 自定义 CSS 能改变 HTML 导出的标题颜色等可视样式
- 缺失外部依赖（如 pandoc）时给出友好提示
