# 默认启动不创建未命名文件 + 空白引导面板

## 需求
1. 全新启动（无会话恢复）时，不自动创建"未命名"空白文件标签，而是显示空白引导面板
2. 只有用户点击 Tab 栏右侧的 "+" 按钮时，才创建新的未命名文件
3. 空白引导面板可显示提示信息或使用说明（如快捷键、打开文件/文件夹入口等）

## 验收条件
- [T-1] 清空配置首次启动，无标签页，显示引导面板
- [T-2] 点击 "+" 按钮后创建未命名文件标签
- [T-3] 会话恢复有文件时正常恢复，不显示引导面板
