# Bug + 改进：快捷键面板视觉统一

## 问题清单
1. **表格选中虚线**：快捷键面板中表格行被选中时，单元格周围出现虚线（焦点矩形 / focus rectangle），视觉杂乱 → 去掉
2. **类别列对齐**：表格"类别"列当前左对齐 → 改为居中对齐
3. **关闭按钮风格不统一**：
   - 快捷键面板（`ShortcutsDialog`）的关闭按钮风格与"更新历史"（`ChangelogDialog`）的关闭按钮不一致
   - "关于"对话框（`AboutDialog`）的关闭按钮也需要一并调整
   - 期望：以"更新历史"对话框的关闭按钮为基准，让三处按钮风格完全一致（位置、尺寸、配色、悬停态）

## 修复目标
- 表格行选中无虚线、键盘焦点用更含蓄的视觉（仅背景色变化）
- "类别"列内容水平居中
- 三个对话框（快捷键 / 更新历史 / 关于）的底部关闭按钮：位置、字号、内边距、圆角、配色、悬停态完全一致

## 验收
- T-1：快捷键面板选中行无任何虚线/焦点框
- T-2：快捷键面板"类别"列文字水平居中
- T-3：三个对话框（快捷键 / 更新历史 / 关于）底部关闭按钮视觉一致，主题切换后仍保持一致
