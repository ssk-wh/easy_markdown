# P3 远期功能 Backlog

## 背景
以下功能属于远期/探索性需求，短期内不会实施，但保留为有意识的待评估项。一旦决定开做，应从本文件拆出独立 Plan 或升级为完整 Plan。

## 动作（候选清单，非按顺序）
- [ ] 实时协作：多人同时编辑同一文档（WebSocket）
- [ ] 版本历史：自动保存编辑历史，支持回退到任意版本
- [ ] Markdown 扩展：数学公式（KaTeX）、流程图（Mermaid）、任务列表
- [ ] 模板系统：内置常用模板（会议记录、技术文档、博客文章）
- [ ] 云同步：WebDAV / Dropbox / OneDrive
- [ ] 插件系统：用户自定义语法高亮、导出格式、快捷键

## 验收
- 任意一项被选中实施前，先从本 Backlog 拆出独立 Plan 并写明 Spec 关联
- 本 Backlog 文件自身不会进入 `completed` 状态，除非全部转出或集体作废
