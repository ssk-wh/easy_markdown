# 清新 / 极简组 3 款主题落地（D/E/F）

## 背景

用户挑选 UI Designer agent 产出的 D Paper Mist / E Mint Breeze / F Monochrome Zen
三个 mockup 全部做成真 TOML 主题，与前 3 款 iOS Liquid Glass 主题并存。

- [x] 生成 3 个 TOML：`resources/themes/paper-mist.toml` / `mint-breeze.toml` / `monochrome-zen.toml`
- [x] `resources/resources.qrc` 加 3 个 theme 资源
