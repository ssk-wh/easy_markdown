# 模块：sync（滚动同步）

## 职责

编辑器与预览之间的双向滚动同步，保证用户在任一侧滚动时另一侧自动对齐到相同的文档位置。

## 对应源码

`src/sync/`

## Spec 清单

| 编号 | 标题 | 状态 | 对应源文件 |
|------|------|------|-----------|
| 01 | 滚动同步 | draft | `ScrollSync.h/cpp` |

## 依赖关系

```
ScrollSync
  ├─ 持有 EditorWidget 指针（不拥有）
  ├─ 持有 PreviewWidget 指针（不拥有）
  ├─ 监听两侧的 scrollValueChanged 信号
  └─ 通过源字符偏移双向换算滚动位置
```

## 性能预算

| 操作 | 目标 |
|------|------|
| 同步延迟 | < 16 ms |
| 同步算法复杂度 | O(log n) |

## 全局约束

- 必须防止信号循环（A → B → A 的回调环）
- 同步应该是「软同步」：允许小幅误差，不强制像素级一致
- 预览内无法定位的源位置（如解析器删除的空行）应 fallback 到最近可见块
