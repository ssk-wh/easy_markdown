# Inline Code Test

## Basic Inline Code

This text contains `inline code` in the middle of the sentence.

Multiple `code` words `can` appear `in` `a` `single` line.

## Inline Code with Special Characters

Some special characters: `@#$%^&*()` should work in inline code.

Unicode characters: `你好世界` and `🎉` should also work.

## Mixed Content

This paragraph has **bold**, *italic*, and `code` all mixed together.

**Bold with `code` inside** should also work correctly.

*Italic with `code` inside* is also valid.

## Code at Line Boundaries

Code at the start: `const x = 5;` is valid.

Code at the end: `let y = 10;`

`Code on its own line.`

## Multiple Code Blocks

First: `function foo()` then `function bar()` and finally `function baz()`.
